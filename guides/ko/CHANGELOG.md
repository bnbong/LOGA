# 업데이트 내역

## 2026년 1월 15일

### ✨ 새로운 기능

#### 1. 우승자 배지 개선

- **변경사항**:

  - 왕관 아이콘(👑) → 트로피 아이콘(🏆)으로 변경
  - 카드의 모서리에 걸치는 형태로 재배치
  - 우승 대회 및 년도 정보를 툴팁으로 표시
  - 골드 그라데이션과 발광 효과 추가

- **효과**:
  - 더욱 명확한 우승자 표시
  - 몰입감 있는 시각적 피드백
  - 마우스 오버 시 자세한 우승 정보 확인 가능

#### 2. 다국어 지원 (한글/영어)

- **기능**:

  - 우측 상단의 언어 전환 버튼으로 한글/영어 변경
  - UI 전체 텍스트 번역 지원
  - 로컬 스토리지에 언어 설정 자동 저장

- **지원 언어**:

  - 🇰🇷 한국어
  - 🇺🇸 English

- **번역 범위**:
  - 버튼 텍스트
  - 우승 대회명
  - 포지션명
  - 커뮤니티 관련 텍스트

#### 3. 공유 스토리지 솔루션

- **문제점 해결**:

  - 기존: 로컬 스토리지만 사용하여 다른 사용자가 공유한 로스터를 볼 수 없음
  - 해결: Firebase 연동 옵션 추가

- **지원 기능**:

  - Firebase Realtime Database/Firestore 연동 지원
  - Fallback으로 로컬 스토리지 사용
  - 중복 좋아요 방지 (사용자 ID 기반)
  - 실시간 동기화 준비

- **설정 방법**:
  - `.env.local.example` 파일을 `.env.local`로 복사
  - Firebase 프로젝트 생성 및 설정
  - 자세한 내용은 [STORAGE_SOLUTION.md](STORAGE_SOLUTION.md) 참조

### 📖 사용 방법

#### 언어 변경

1. 우측 상단의 언어 버튼 클릭
2. 한글(🌍 EN) 또는 영어(🌍 한글) 선택
3. 자동으로 페이지가 새로고침되며 언어 적용

#### Firebase 설정 (선택사항)

1. [STORAGE_SOLUTION.md](STORAGE_SOLUTION.md) 가이드 참조
2. Firebase 프로젝트 생성
3. `.env.local` 파일에 설정 추가
4. 앱 재시작

### ⚠️ 주의사항

- Firebase를 설정하지 않으면 로컬 스토리지가 사용됩니다
- 로컬 스토리지는 브라우저별로 독립적입니다
- 언어 변경 시 페이지가 새로고침됩니다
